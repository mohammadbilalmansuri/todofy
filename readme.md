# Todofy – Where Todos Take Shape

Todofy is the ultimate task management app, designed to help you prioritize and organize your daily tasks with ease. Create, update, and delete tasks seamlessly, then mark them as completed to track your progress. With its sleek, intuitive design, Todofy keeps you focused and motivated to achieve your goals every day.

## Features

- **User Authentication**: Sign up, log in, log out, and delete your account.
- **Task Management**: Quickly create, update, and delete tasks.
- **Task Completion**: Easily track progress by marking tasks as completed.

## Technologies Used

### Frontend

- HTML
- Tailwind CSS
- JavaScript

### Backend

- Node.js
- Express.js
- MongoDB with Mongoose
- JWT (JSON Web Tokens)
- bcrypt
- Zod
